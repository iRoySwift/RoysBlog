## 课后练习

通过命令行，发行一个代币。并给自己账号 mint 一定数量的代币。 并通过插件钱包或者命令行的方式给其他同学空投该代币

提示：

-   命令行操作，查看 spl-token 命令的帮助文档

```bash
solana config set --url https://api.devnet.solana.com
```
